# Таможенный помощник — лендинг
Готовый одностраничный сайт для GitHub Pages.

## Быстрый старт
1. Создайте публичный репозиторий на GitHub (например `customs-helper`).
2. Загрузите файлы из этого архива в корень репозитория.
3. Включите GitHub Pages: Settings → Pages → Source: *Deploy from a branch*, Branch: *main*, Folder: */ (root)*.
4. Готово! Сайт станет доступен по адресу `https://<username>.github.io/customs-helper/`.

### Что можно настроить
- Замените ссылку на бота в `index.html` (поиск по `yourbotlink`). 
- Обновите контакты и тексты.
- Поместите ваши документы (оферта/политика) и замените заглушки ссылок в футере.
